# Bioinformatics Portfolio

This repository contains a collection of bioinformatics projects focused on:

- 🧬 Single-cell RNA sequencing (scRNA-seq)
- 🧪 Biomarker discovery
- 🧫 Cell & gene therapy analytics

## Project List

| Project | Description | Status |
|--------|-------------|--------|
| `scRNAseq-PBMC` | End-to-end single-cell analysis using 10X PBMC data | 🔧 In progress |
| `bulkRNAseq-Biomarkers` | Prognostic biomarker discovery using TCGA breast cancer | ⏳ Planned |
| `celltherapy-response` | Signature analysis in CAR-T therapy responders | ⏳ Planned |

## Technologies

- Python, R, Seurat, Scanpy, DESeq2, ggplot2
- GEO, TCGA, Bioconductor
- Git, GitHub, Streamlit
